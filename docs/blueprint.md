# **App Name**: LinkLook

## Core Features:

- URL Input: Allow users to input a URL.
- Content Fetching: Fetch the content of the given URL.
- Title & Description Extraction: Extract the title and description from the fetched content.
- Image Selection Tool: AI powered tool which analyzes the content to suggest relevant preview images and selects the best one.
- Preview Generation: Generate a preview with title, description, and image.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5), conveying trust and reliability for online linking.
- Background color: Very light blue (#E8EAF6). It gives enough visual contrast to work with dark-themed primary blue while providing comfortable experience.
- Accent color: Purple (#9575CD) which can act as a secondary call to action, working in coordination with the blue without disrupting.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text. It creates the balance between stylish readability and clear concise text.
- Clean, card-based layout to display the link preview prominently.
- Simple, outlined icons for actions like copy or share.
