# **App Name**: MetaGrab

## Core Features:

- URL Input: Allow users to input a URL.
- Metadata Fetch: Automatically fetch title, description, and images.
- Thumbnail Detection: Intelligently selects and displays the main thumbnail image.
- Social Media Profile Detection: Identify and display social media profile images associated with the URL using a tool that analyzes site structure and content.
- Metadata Display: Clearly display the fetched metadata in a structured format.
- Copy Functionality: Enable users to copy specific metadata fields (title, description, images).

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to provide a sleek and sophisticated feel.
- Background color: Very light gray (#F0F0F0) to create a clean and unobtrusive backdrop that lets the purple stand out.
- Accent color: A lighter purple (#9575CD) to highlight interactive elements, and to maintain an analogous color scheme.
- Font: 'Inter' sans-serif for both headlines and body text. The font provides a clean, modern and readable feel suitable for a utility application.
- Use simple, outline-style icons to represent actions such as copying metadata, keeping with the app's sleek aesthetic.
- Employ a clean, card-based layout to organize metadata fields. Ensure responsive design for various screen sizes.
- Use subtle animations for loading states and copy confirmations to enhance the user experience without being distracting.
